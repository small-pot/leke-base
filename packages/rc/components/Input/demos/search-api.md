---
title: Input.Search API
---

| 参数 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| defaultValue | 默认值 | boolean | - |
| value | 当前值 | boolean | - |
| size | 大小 | 'middle' \| 'large' | 'middle' |
| enterButton | 搜索按钮 | ReactNode | - |

<br>
Input.Search 的其他属性和 Input 一致。
