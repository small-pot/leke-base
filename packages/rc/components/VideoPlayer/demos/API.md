---    
title: API
description: 实列会暴露挂载的Video Element，可以取到所有的原生属性。之外，提供额外的属性
---
| 属性 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| el | 挂载实例 | element | - |
| src | 资源路径 | string | - |
| width | 宽度 | number | 638 |
| height | 高度 | number | 358 |
| autoplay | 自动播放 | boolean | false |
| loop | 循环播放 | boolean | false |
| poster | 视频封面 | string | - |
| muted | 静音 | boolean | false |
